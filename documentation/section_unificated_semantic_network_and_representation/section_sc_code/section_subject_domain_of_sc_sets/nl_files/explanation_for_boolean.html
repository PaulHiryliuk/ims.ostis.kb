булеан* - это бинарное отношение , каждая пара которго связывает некое множество si (указываемое в качестве второго компонента) и множество sj, которое является семейством всевозможных подмножеств множества si, которое выполняет роль перевого компонента компонента. Указанное направление данной упорядоченной пары выбрано для того, чтобы подчеркнуть связь отношения булеан* с отношением объединение*
