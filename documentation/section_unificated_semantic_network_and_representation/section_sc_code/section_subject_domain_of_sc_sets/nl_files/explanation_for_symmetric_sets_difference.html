симметрическая разность множеств* - это бинарное отношение между множеством si (указываемое в качестве второго компонента) и парой множеств sj и sk (указываемой в качестве первого компонента). Множество si есть результат симментрической разности множества sj и sk в том и только том случае, когда любой элемент множества si принадлежит либо множеству sj, либо множеству sk, но не принадлежит обоим множествам одновременно.
