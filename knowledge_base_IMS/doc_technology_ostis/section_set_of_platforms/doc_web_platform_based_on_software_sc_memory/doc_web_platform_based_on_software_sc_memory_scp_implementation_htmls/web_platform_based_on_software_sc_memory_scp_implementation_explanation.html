Реализация scp-машины в web-ориентированной платформе интерпретации sc-моделей интеллектуальных систем, основанной на специальном формате кодирования sc-текстов базируется на соответствующей Реализации sc-памяти в web-ориентированной платформе интерпретации sc-моделей интеллектуальных систем, основанной на специальном формате кодирования sc-текстов. Для реализации использованы языки программирования C и C++.

Указанная реализация разделяется на две основных части:
  1. Библиотека процедур, реализующих операционную семантику scp-операторов и содержащую объявления используемых внутренних структур.
  2. Библиотека реализаций атомарных sc-агентов, описанных во введении в раздел унифицированные логико-семантические модели машин обработки знаний.

Такое разделение позволяет реализовывать различные подходы к интерпретации scp-программ (т.е. реализации абстрактной scp-машины), не изменяя при этом часть, реализующую логику непосредственно самих scp-операторов.
