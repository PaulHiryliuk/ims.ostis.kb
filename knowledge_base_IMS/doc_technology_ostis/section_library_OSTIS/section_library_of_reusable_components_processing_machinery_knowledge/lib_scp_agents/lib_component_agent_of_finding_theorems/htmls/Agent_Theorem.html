Задачей агента поиска теорем для заданной онтологии является нахождение теорем, относящихся к заданной онтологии. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска теорем для заданной онтологии. Аргументом запроса является онтология , в рамках которой необходимо найти все теоремы.
