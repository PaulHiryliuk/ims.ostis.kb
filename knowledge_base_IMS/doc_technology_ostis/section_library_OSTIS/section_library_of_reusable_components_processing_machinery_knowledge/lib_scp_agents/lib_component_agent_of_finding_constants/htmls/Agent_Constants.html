Задачей агента поиска поиска понятий, на основе которых определяется заданное понятие является нахождение понятий, на основе которых определяется заданное понятие. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска понятий, на основе которых определяется заданное понятие. Аргументом запроса является понятие, для которого определяются константы.
