Задачей агента поиска доказательства для заданного утверждения является нахождение доказательства для заданного утверждения. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска доказательства для заданного утверждения. Аргументом запроса является утверждение, для которого необходимо найти доказательство.
