Задачей агента поиска аксиом для заданной онтологии является нахождение аксиом, относящихся к заданной онтологии. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска аксиом для заданной онтологии. Аргументом запроса является онтология, в рамках которой необходимо найти все аксиомы.
