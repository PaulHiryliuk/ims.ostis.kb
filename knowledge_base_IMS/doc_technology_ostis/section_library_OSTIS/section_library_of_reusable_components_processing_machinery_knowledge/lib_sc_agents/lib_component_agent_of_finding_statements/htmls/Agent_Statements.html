Задачей агента поиска утверждений для заданного понятия является нахождение утверждений для заданного понятия. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска утверждений для заданного понятия. Аргументом запроса является понятие, для которого необходимо найти утверждения.
