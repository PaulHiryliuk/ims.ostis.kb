Задачей агента поиска решения задачи заданного типа является нахождение решения задачи заданного типа. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу решения задачи заданного типа. Аргументом запроса является задача определённого типа, для которой необходимо найти решение.
