Задачей агента поиска иллюстраций для заданного понятия является нахождение иллюстраций для заданного понятия. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска иллюстраций для заданного понятия. Аргументом запроса является понятие, для которого необходимо найти изображения.
