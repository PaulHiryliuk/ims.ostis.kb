Задачей агента поиска определения/пояснение для заданного понятия является нахождение определений для заданного понятия. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска определения/пояснения для заданного понятия. Аргументом запроса является понятие, для которого необходимо найти определение либо пояснение.
