Задачей агента поиска ключевых понятий для заданной предметной области является нахождение ключевых понятий для заданной предметной области. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска ключевых понятий для заданной предметной области. Аргументом запроса является предметная область, для которой необходимо найти ключевые понятия.
