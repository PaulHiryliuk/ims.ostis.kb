Задачей scp-программы поиска общих дуг для двух множеств является поиск общих дуг для двух заданных множеств. Первым и вторым входными параметрами данной scp-программы являются два анализируемых множества, третьим входным параметром является множество дуг одного направления, связывающих два рассматриваемых множества, четвёртым входным параметром является узел ответа, куда, в конечном счёте, будут занесены все дуги с соответствующими им атрибутами или неролевыми отношениями.
