Задачей scp-программы поиска общих элементов для двух заданных множеств является поиск общих элементов для двух заданных множеств. Первым и вторым входными параметрами данной scp-программы являются два анализируемых множества, третьим входным параметром является множество, куда будут занесены общие для двух множеств элементы.
